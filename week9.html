Bootstraps

Created out of boredom by developers who wanted a faster way to create UI and resolve all nook and crannies imposed by different browsers. (lol)

Boostrap uses: Flexbox grid for layout. rem for measurements. Device breakpoints to accomodate for diff media sizes.

Bootstrap Grid system helps you to create your layout and easily build a responsive website. EG (imagine <> brackets as []):
[div class="container"] (provides a means to center and horizontally pad content of site)
[div class="row"] (wrappers for columns. each has horizontal padding for controlling space in between)
[div class="col-sm"] (grid columns without specified width will automatically be layout as equal width columns. column classes indicate number of columns to be used out of 12. EG: .col-4 gives 3 equal-width columns)
One of three columns
[/div]
[div class="col-sm"]
One of three columns
[/div]
[div class="col-sm"]
One of three columns
[/div]
[/div]
[/div]

Responsive img - use .img-fluid with max-width:100% and height:auto for images which scale with parent. Image thumbnails - use .img-thumbnail to create an image with a 1px border.

Other components that boostraps can change: Alerts, Forms, Navbar, Video Embeds.
