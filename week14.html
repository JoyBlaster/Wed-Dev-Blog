Lottie Animations

Lottie was made by some people at AirBNB to make animations easier to process and make.

Lottie animations are lightweight animations and design since its just code.

  1. Graphics prepped with Illustrator/other 3rd party tools.
  2. Animation created in AfterEffects
  3. Export JSON file with bodymoving plugin
  4. Upload into Lottie for further animation making

LottieFiles can be found online or be made.

LottieFiles can be made with code.

SVG files can be converted to LottieFiles.
